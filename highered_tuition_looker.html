How it works

A 5-step research pipeline, not a database lookup

Every search triggers a live AI pipeline that reads the actual university website — not cached data from a third party.

1

Web search

Finds 3–5 candidate tuition URLs on the university's official .edu domain, prioritising program-specific pages and PDF cost sheets.

2

Page scraping

Reads the live content of the tuition schedule page, extracting exact dollar figures and academic year.

3

Program page

When a program is specified, a second dedicated page is fetched — MBA, law, nursing and other programs often have different rates.

4

PDF reading

Bursar PDFs and cost-of-attendance documents are fetched and read directly — not just linked.

5

Synthesis & fact-check

Multiple sources are cross-checked for consistency. Discrepancies are flagged. The most specific, most recent data wins.

Features

Everything you need for tuition research

Built for students, advisors, researchers, and anyone who needs accurate, citable tuition data.

Single university lookup

Search any US university by name with an optional program and level filter. Results include source links, confidence score, and notes.

Program-specific tuition

MBA, JD, MD, engineering, nursing — specify a program name to retrieve rates specific to that programme, not just general university fees.

Total credits required

Where available, pulls the total credit hours required to complete the programme — useful for estimating total degree cost.

Anti-hallucination guarantee

The AI must quote verbatim text from the source page for every dollar figure. If no explicit figure is found, the field is null — never guessed.

Bulk Excel upload

Upload an Excel file with university names, programs and levels. The tool processes all rows and exports a formatted .xlsx report with all results.

Verified source links

Every result includes the exact URL of the official university page or PDF where the data was found — HTML and PDF sources are labelled separately.
